= freeplay
Simple rest client for Freeplay

== Usage

=== Installation

To install, in Gemspec

gem 'freeplay'

=== Initialize
  @freeplay = Freeplay.new(api_token, api_key, app_id, device_type(optional))

=== Register user
  @freeplay.get_user(user_id, id_type(optional))

=== Check for reinforcement
  @freeplay.reinforce?(user_uri, hablet_id)

=== Redeem
  @freeplay.redeem(user_uri, hablet_id, email)
